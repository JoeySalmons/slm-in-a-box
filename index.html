slm-in-a-box

An experiment in running small language models locally with minimal dependencies. A 50M parameter language model is packed directly into a single HTML file that runs in your web browser.

Download

Save the file locally and open it directly in a modern web browser. No installation, server, or internet connection is required after downloading.

Get supra50m_chat-v2.html (~120 MB)

Works on desktop and mobile platforms with compatible modern browsers.

Features

  • Fully Offline: Run inference locally from a USB stick or hard drive after initial download.
  • Configurable Settings: Adjust temperature, top-p, top-k, max generation tokens, and repetition penalty.
  • Interactive Chat Interface: Edit the conversation history directly, configure system prompts, and use copy/regenerate controls.
  • Local Performance: Achieves around 50 tokens per second on a modern laptop using CPU-based WebAssembly.
Technical Note: The experimental WebGPU backend option may/will cause the browser tab to freeze. The default CPU (WASM) backend is recommended for stable performance.

Building From Source

To compile the HTML file yourself, you will need the model and tokenizer configuration files from Hugging Face:

  • model_int8.onnx, generation_config.json, and config.json from onnx-community/Supra-1.5-50M-Instruct-exp-ONNX
  • tokenizer.json from SupraLabs/Supra-1.5-50M-Instruct-exp

Place these files in the same directory as the build script and execute:

python build_supra_chat-v2.py

This generates the self-contained HTML file containing the model weights and the interface code.
